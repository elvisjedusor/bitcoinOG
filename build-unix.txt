UNIX BUILD NOTES (Ubuntu 24.04 LTS)
===================================

This codebase has been modernized to compile on Ubuntu 24.04 LTS with:
- GCC 11+ / C++11
- OpenSSL 3.x (with compatibility wrappers)
- Berkeley DB 5.3 (using C API compatibility layer)
- Boost 1.74+

Dependencies
------------
sudo apt-get update
sudo apt-get install build-essential
sudo apt-get install libssl-dev
sudo apt-get install libdb-dev libdb5.3-dev
sudo apt-get install libboost-all-dev

For GUI builds (optional, Phase 2):
sudo apt-get install libwxgtk3.2-dev
sudo apt-get install libgtk-3-dev


Building the Daemon (bitcoinogd)
--------------------------------
make -f makefile.unix clean
make -f makefile.unix

The daemon binary will be created as 'bitcoinogd'.


Building the GUI (bitcoinog) - Phase 2
--------------------------------------
Requires wxWidgets. After installing libwxgtk3.2-dev:
make -f makefile.unix bitcoinog


Installation
------------
sudo make -f makefile.unix install

This installs bitcoinogd to /usr/local/bin/


Running
-------
# Create data directory
mkdir -p ~/.bitcoinog

# Run daemon
./bitcoinogd

# Run with options
./bitcoinogd -daemon          # Run in background
./bitcoinogd -gen             # Enable mining
./bitcoinogd -rpcuser=user -rpcpassword=pass


Troubleshooting
---------------
If you encounter OpenSSL errors about deprecated functions, ensure
OPENSSL_SUPPRESS_DEPRECATED is defined in the makefile (already set).

If Berkeley DB errors occur, the code now uses a C API compatibility
layer (db_cxx_compat.h) instead of the deprecated C++ bindings.


Modernization Notes
-------------------
The following changes were made for Ubuntu 24.04 compatibility:

1. OpenSSL 3.x: BIGNUM is now opaque, CBigNum class refactored to wrap
   BIGNUM* instead of inheriting from BIGNUM struct.

2. Berkeley DB: C++ bindings (db_cxx.h) removed from modern Ubuntu.
   Created db_cxx_compat.h wrapper using C API.

3. Boost: Added BOOST_BIND_GLOBAL_PLACEHOLDERS to suppress deprecation
   warnings about bind placeholders.

4. GCC 11+: Fixed narrowing conversions, added C++11 standard flag.

5. wxWidgets: Daemon build no longer requires wxWidgets. GUI-specific
   code is conditionally compiled with #if wxUSE_GUI.
